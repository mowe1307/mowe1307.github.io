Challenge 5: Accessibility Concerns

Challenges 2-4 present a number of accessibility concerns. All challenges would present an issue because there are no text-to-speech
options so if someone using my sight has visual impairments, it would be difficult if not impossible to use. Challenge 3 requires typing
and clicking and is not very accessible to people who may have mobility difficulties. To address these concerns, I should create more
text-to-speech options so that I can be more accessible to a wider audience. I might also consider creating a better layout, having better
background and text colors and sizes to make sure it is easy to read and pick through.
All this information is found at Why Accessibility Matters
